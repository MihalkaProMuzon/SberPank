Сберпанк – Сбер Парсер Анкет [ Версия 1.0 ]

Утилита для выгрузки информации из .eml файлов в таблицу Exel.

----------------------------------------
Инструкция к употреблению:
----------------------------------------

Шаг 1. Откройте .еml файл
	Это можно сделать, нажав на кнопку "Открыть файл" в блоке "Парсинг",
	или перетащив нужный файл в окно программы.
	.eml файл будет открыт в новом окне программы.

Шаг 2. Определите поля для парсинга
	Открытый файл представляет из себя преобразованные в строки текстовая и html части .eml,
	именно в таком виде информация, будет считываться программой при парсинге.
	Чтобы добавить поля для парсинга (будущие ячейки таблицы Exel), нужно нажать на кнопку "Добавить поле" 
	и определить параметры поиска информации.

	Параметры поля:
	- Имя: просто условное обозначение поля, не влияет на парсинг.
	- Поиск от: отрывок .eml файла, после которого находится нужная информация поля.
	- Поиск до: отрывок .eml файла, до которого будет выгружена информация поля.
	- Индекс: если отрывок "Поиск от" встречается в .eml файле несколько раз, определяет его номер начиная с 1го.
	- Смещение: Кол-во символов, которое нужно пропустить после завершения отрывка "Поиск от".
		*Используется, например, для пропуска переноса строки (который занимает 2 символа).
	- Повторы: если файл с таким полем уже встречался (например, номер телефона или имя), 
		это будет обозначено в Exel файле выделением соответствующей ячейки.
	- Не обязательное: если отрывки "Поиск от" и "Поиск до" не будут найдены, поле будет просто пропущено.
	- Символ: Номер ячейки Exel таблицы по горизонтали.

	Удалить поле можно нажав на крайнюю правую кнопку "x".

	Получившийся шаблон парсинга можно сохранить для последующего использования.
	Для этого предусмотрены кнопки "Сохранить шаблон" и "Загрузить шаблон".
	Загрузить шаблон можно также перетащив файл .pank в окно программы

Шаг 3. Определите рабочую папку и выходной файл
	Для запуска парсинга необходимо указать папку с файлами .eml, а также выходной .xlsx файл.
	Сделать это можно нажав на соответствующие кнопки "Обзор" в верхней части программы,
	или же перетащив файл Exel и папку в окно программы.
	!! Внутри рабочей директории должны быть только .eml файлы.

Шаг 4. Запуск парсинга
	После установки шаблона полей для парсинга, рабочей директории и выходного файла можно запустить выгрузку.
	Перед запуском можно установить стартовую строку для записи (с какой строки начинать запись в файл Exel),
	а также включить/выключить опцию формирования логов парсинга.
	
	Если в момент записи информации в выходной файл, он будет отрыт в сторонней программе (Exel, к примеру),
	данные выгрузятся в его копию.

	Для начала парсинга нужно нажать кнопку "Начать выгрузку", после этого начнется процесс обработки информации.
	Парсинг завершится советующим уведомлением. Процесс можно прекратить, нажав на кнопку "Прервать".
	
	После завершения выгрузки информации, файлы, где парсинг прошел успешно, 
	будут перемещены в подкаталог "Парсинг успешен". 
	Файлы с неудавшейся выгрузкой информации, будут перемещены в "Парсинг провален".

Шаг 5. Готово
	Выходной файл можно открыть и проверить корректность считывания данных.
----------------------------------------
